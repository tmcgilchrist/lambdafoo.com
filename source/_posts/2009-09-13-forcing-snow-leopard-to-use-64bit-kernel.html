--- layout: post title: Forcing Snow Leopard to use 64bit Kernel tags: - OS X status: publish type: post published: true meta: _edit_last: "1" --- How to force Snow Leopard to boot the 64bit kernel.
  1. Launch TextEdit
  2. Open the file named /Library/Preferences/SystemConfiguration/com.apple.Boot.plist
  3. Add arch=x86_64 to the Kernel Flags string:
    4. Kernel mach_kernel Kernel Flags arch=x86_64
This is posted just about everywhere, and is really only added here for my convenience.
